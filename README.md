# Smart Complaint Management System

## 📌 Project Description
A web-based complaint management system that allows users to submit complaints and helps administrators manage and track them efficiently.

## ✨ Features
- Submit complaints
- View all complaints
- Search complaints
- Filter complaints
- Update complaint status
- Delete complaints

## 🛠 Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the backend:
   node server.js
4. Start the frontend:
   npm start
